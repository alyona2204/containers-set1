# Домашнее задание к лекции «Контейнеры» — Set

## Описание
В рамках данного задания реализована логика формирования команды персонажей с использованием контейнера `Set`.  
Цель — исключить возможность добавления одного и того же персонажа в команду несколько раз.

## Реализация
Создан класс `Team`, который хранит участников команды в поле `members` типа `Set`.

### Методы класса:
- `add(character)` — добавляет персонажа в команду.  
  Если персонаж уже присутствует в команде, генерируется ошибка.
- `addAll(...characters)` — добавляет произвольное количество персонажей в команду.  
  Дубликаты игнорируются, ошибка не выбрасывается.
- `toArray()` — преобразует `Set` с участниками команды в массив.

## Тестирование
Для всех методов класса написаны unit-тесты с использованием Jest.  
Тесты обеспечивают 100% покрытие кода по строкам.

## Используемые технологии
